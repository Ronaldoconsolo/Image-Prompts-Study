# 🎨 Image Prompts Study
 
Este repositório documenta meus estudos com **Stable Diffusion em estilo anime**.  
Rotina: **1h30 por dia**, evoluindo de prompts básicos até avançados, com desafios semanais para consolidar o aprendizado.

---
